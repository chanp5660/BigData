# 이 데이터가 유용한 이유
1. 하루 이용자 건수가 많고 게임도 다양하다.
2. 원래는 현재 상태의 게임시세만 알 수 있다. (그동안의 시세를 보고 판단할 수 있는 데이터가 없다.)
3. 이 데이터를 따라서 게임시세변동을 알 수 있다.
4. 현재가 최저가인지 최고가인지 판단할 수 있게 된다.
